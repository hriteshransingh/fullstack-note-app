# 📝 NoteFlow - Full-Stack Note Management App


A modern note-taking application with secure authentication, real-time CRUD operations, and a responsive UI.

## 🚀 Features

### 🔐 Authentication
- JWT-based login/signup
- Cookie-based session handling
- Protected routes

### 📋 Note Management
- Create, Read, Update, Delete notes
- Instant save functionality

### 🎨 UI/UX
- Responsive design (mobile + desktop)
- Tailwind CSS with custom animations
- Toast notifications

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Hooks
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication


## 🛠️ Setup .env file

Before running the project, create a `.env` file in the root directory and add the following:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=3000
JWT_SECRET=your_jwt_secret
NODE_ENV=development
```


## ⚙️ Build the App
```
npm run build
```


## 🚀 Start the App
```
npm start
```
